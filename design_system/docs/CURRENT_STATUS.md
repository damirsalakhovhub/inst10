# Текущее состояние проекта

## На чем остановились

Работаем над системой дизайн-токенов для компонента кнопки. Реализована полная система семантических токенов для всех размеров кнопок (tiny, small, medium, large, xlarge), включая размеры иконок (`--btn-*-icon-size`) и отступы между иконкой и текстом (`--btn-*-gap`). Все стили кнопок вынесены в модульную структуру: базовые стили, размеры, варианты цветов, состояния и иконки. Настроен watch-режим для SCSS с автоматической пересборкой при изменении токенов (используется `sass --watch --poll --load-path=app`). Создан универсальный класс `.ui-kit__row` для выстраивания элементов в ряд без растягивания дочерних элементов. UI Kit страница для кнопок отображает все варианты размеров, цветов и комбинаций с иконками. Следующий шаг — продолжить работу над другими компонентами UI Kit или доработать систему токенов для кнопок.
