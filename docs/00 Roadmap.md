MVP

1) Фундамент проекта
Модели/миграции, минимальная навигация, базовая структура UI.
[Foundation](./01_Foundation.md)

2) Проекты
Создание/редактирование проекта, список проектов, переход в проект.

3) Задачи v1
CRUD задачи (title, description), статусы (ToDo/InProgress/Done), приоритет, due/start даты.

4) Карточка задачи
Единый экран: описание, свойства, список вложений, комментарии.

5) Черновики задач
Задача видна только автору до публикации (private draft).

6) Поиск и фильтры
Полнотекст по title/description, фильтры по статусу/датам/приоритету.

7) Сохранённые представления
"Сегодня", "Просрочено", "Черновики", "На неделе" + кастомный фильтр.

8) Вложения v1
Загрузка/удаление/просмотр списка вложений у задачи (локальные).

9) Встроенный просмотр PDF
Открытие PDF внутри карточки задачи без скачивания.

10) Превью таблиц
Просмотр CSV/XLSX (первые N строк, листы) внутри карточки задачи.

11) Упоминания @
Распознавание @имя в тексте/комментариях, автоссылка на пользователя (MVP-уровень, пока только "я").

12) Roadmap v1
Таймлайн по задачам проекта (start/due), без зависимостей (минимальный Gantt).

13) Календарь нерабочих дней
Настройки "нерабочие даты" и учёт их в Roadmap (ручной).

14) Быстрый ввод
Создание задачи одной строкой + горячие клавиши (минимум).

15) Стабилизация
Валидации, обработка ошибок, тесты на ключевые потоки (smoke), минимальная безопасность для загрузок (лимиты, типы).
